# Personal Event Planner App

## Overview
This is an Android application that allows users to manage personal events. Users can create, view, update, and delete events easily.

## Features
- Add new events
- View all events in a list
- Update existing events
- Delete events using swipe gesture
- Date and time picker for event scheduling

## Technologies Used
- Java
- Android Studio
- Room Database
- RecyclerView
- Fragments

## How to Use
1. Click "Add Event" to create a new event
2. Fill in event details and select date/time
3. Click on an event to update it
4. Swipe left/right to delete an event
